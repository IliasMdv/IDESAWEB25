# Sitio Web Oficial del Club Deportivo Montepalma
## Descripción del proyecto
Este proyecto consiste en el desarrollo de un sitio web para el Club Deportivo Montepalma, un **centro deportivo que ofrece instalaciones** de tenis, pádel, fútbol, piscina y fitness. 
La web servirá como plataforma para mostrar horarios, tarifas, noticias, eventos, y permitir la reserva online de pistas y clases.

El proyecto será desarrollado con tecnologías web modernas y contará con un panel administrativo para que el club pueda gestionar su contenido sin depender de terceros.

> **"Digitalizar la experiencia deportiva para conectar mejor con socios y atraer nuevos clientes."**

## Público objetivo

## Público objetivo
- Socios actuales del club que necesiten información actualizada y reservar pistas.
- Potenciales clientes interesados en actividades deportivas en la zona.
- Personal administrativo del club que gestionará contenidos y reservas.
- Entrenadores y monitores que publicarán horarios y noticias.

## Funcionalidades principales
- **Página de presentación** del club y sus servicios.
- **Sistema de registro e inicio de sesión** para usuarios.
- **Reserva online** de pistas y clases.
- **Calendarios de eventos y torneos.**
- **Panel administrativo** para gestión de contenidos.
- **Formulario de Contacto** integrado con Google Maps
- **Blog o sección de noticias del club.**

## Tecnologías previstas
A continuación se listan las principales herramientas y tecnologías a utilizar:

### Frontend
- HTML5 semántico.
- CSS3 con Bootsrap o Tailwind.
- JavaScript para interactivad.
- Diseño responsive y accesible.

### Backend
- PHP 8 para la lógica del servidor.
- MySQL para la base de datos.
- Autenticación y gestión de sesiones.
- API REST para comunicación frontend-backend.

Otras tecnologías:
- **Control de versiones:** _Git y GitHub_  
- **Servidor local:** _XAMPP _ 
- **Diseño de prototipos:** _Figma _ 
- **Librerías JS:** _FullCalendar, SwiperJS._
- **APIs externas:** _Google Maps,_ posible integración con redes sociales
